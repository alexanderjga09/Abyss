# 🐟 Abyss

## 📋 Prerequisites

- For the bot to work correctly

```bash
py -3.13 -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### Clone the repository

```bash
git clone https://github.com/alexanderjga09/Abyss.git
cd Abyss
```

### Token for the bot

- Create a `.env` file and write the following inside it

```bash
TOKEN="BOT_TOKEN"
```
